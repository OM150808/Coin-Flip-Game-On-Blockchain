# Coin Flip

A decentralized coin flip game built on the Stellar Testnet. This project uses React, Vite, and the Freighter wallet extension to provide a seamless blockchain gaming experience.

## Project Description

Stellar Coin Flip is a fast-paced, "winner-stays-put" game where users can bet their Testnet XLM on a coin toss. 
- **Connect Wallet**: Securely connect using the Freighter wallet extension.
- **Dynamic Odds**: Experience a provably fair simulation on the Stellar network.
- **Transaction History**: Track your flips and results directly on the blockchain.
- **Loss Deduction**: Only losing flips result in a transaction to the "house" address, keeping winning streaks efficient.

## Setup Instructions

To run this project locally, follow these steps:

### Prerequisites
- Node.js (v16+)
- npm or yarn
- [Freighter Wallet](https://www.freighter.app/) extension installed in your browser.

### Installation

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd Coin-Flip-Game-On-Blockchain
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Access the application**:
   Open [http://localhost:5173](http://localhost:5173) in your browser.

### Configuring Freighter
- Ensure your Freighter wallet is set to **Testnet**.
- Make sure you have some Testnet XLM use the [Stellar Laboratory Friendbot](https://laboratory.stellar.org/#account-creator?network=testnet).
